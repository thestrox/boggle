# README

This is a boggle game developed using Ruby on Rails and React.

* System dependencies
    * Ruby 2.6.5
    * Rails 6.0.2
    * Node 12.13.0
    * Yarn 1.22.0
    * React 16.12.0

* Configuration

    Run the application using following series of commands:

    ```
    gem install bundler
    bundle install
    yarn install
    rails s
    ```

* How to run the test suite
    * Run Rails test using
        ```
        rails test
        ```
    * Run React test using
        ```
        yarn test
        ```
